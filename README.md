# Guardian Shield - AI-Based Malicious Content Detection System

## Project Overview

Guardian Shield is a real-time mobile security application that uses AI/ML to detect and block malicious images, links, and QR codes shared on social media platforms.

## Features

- Real-time content scanning
- Multi-layer threat detection (URLs, Images, QR Codes)
- Machine Learning-powered classification
- On-device processing for privacy
- User-friendly warning interface

## Tech Stack

- **Android**: Kotlin, Jetpack Compose, Room
- **ML**: TensorFlow Lite, Scikit-learn, PyTorch
- **APIs**: Google Safe Browsing, VirusTotal

## Getting Started

### Prerequisites

- Android Studio Arctic Fox or later
- Python 3.8+
- Git

### Setup

1. Clone this repository
2. Open android-app in Android Studio
3. Setup Python virtual environment
4. Install dependencies: `pip install -r ml-models/requirements.txt`
